1/48 Sikorsky H-19 Chickasaw/S-55 Whirlwind Cockpit Detail Set
$25.00 plus shipping
Item # SW006

       Designed for the Revell 1/48 H-19 'Rescue Helicopter', also suitable for the various licensed reissues in Japan of the S-55 and H-19 dating from 1955 through today. This set is also compatible with the Revell 1/48 S-55 Whirlwind kit. The Whirlwind is from the same masters as the H-19. This set is specifically designed to improve the flight deck of this model by replacing the horribly inaccurate seats, instrument panel and center console. Let me qualify everything I'm saying here by stating that I am not a professional mold maker, pattern maker or resin caster. I simply strove to repair the inadequacies of this kit for my own pleasure and in doing so ended up with some masters and molds enabling me to share this with other interested modelers. Look carefully at the pictures and see just what you will be getting with this package. First you will receive 2 flight seats consisting of a seat back and a seat base both with molded in place harnesses, next is a more accurate center console and a well detailed instrument panel. I used several pictures of actual aircraft instrument panels to create this and while it is not an exact replica of any one aircraft it is more of a composite of commonly seen details. It is detailed on both the front and back sides which is important because on the real aircraft the back side of the IP was highly visible. Additionally you will receive a pair of collectives which the base kit lacks entirely. Also included is a single sheet of instructions for assembly and modification of the kit floor-pan.
       You are purchasing an unassembled, unpainted detail set. The finished pictures are only to show you what is possible with this package. These are all 'garage cast' pieces so some clean-up and trim work will be required and surgery to the donor kit is necessary. This is a resin detail set intended for advanced modelers only and will require the use of superglue to complete. I admit that this does not produce a 100% accurate H-19 cockpit but it does produce something dramatically better than what the kit offers.










1/48 Sikorsky H-19 Chickasaw/S-55 Whirlwind Clear Parts Set
$10.00 plus shipping
Item # SW007

       Designed for the Revell 1/48 H-19 'Rescue Helicopter', also suitable for the various licensed reissues in Japan of the S-55 and H-19 dating from 1955 through today. This set is also compatible with the Revell 1/48 S-55 Whirlwind kit. The Whirlwind is from the same masters as the H-19. This set was designed with the intent to remedy several issues existing within the original 1955 molds for the Revell 1/48 H-19 Rescue Helicopter or the Revell S-55 Whirlwind. Specifically the original kit pieces have a very loose fit to the model and are so thick that they resemble coke bottle glass. The kit also does not provide any clear panes for the cargo compartment. This vacu-formed set of clear parts includes a slightly larger windshield that better fits the kit fuselage, new side panes that are flush for the two aft panes yet retain the blown door panes. The set also includes four new panes for the cargo bay area. Also included are two complete flight deck doors that can be used to replace the kit molded closed doors - this will allow you to build your model with doors in the open position. This set comes with a single sheet of usage instructions.





1/48 Sikorsky H-19 Chickasaw/S-55 Whirlwind Cargo Bay Detail Set
Price $35.00
Item # SW008

       The cargo bay detail set provides a new front and aft bulkhead, canvas and web seats set for use on the left side and stowed on the right side. Also with this set are two pieces for the power transmission housing and a fire extinguisher for the aft bulkhead. The set includes cutting templates for the floor, ceiling and a forward bulkhead to blank off the area ahead of the flight deck instrument panel. Lastly the set includes a new resin side door so the modeler can open things up to see the interior. I do not claim that this set is 100% accurate however it does replicate some of the more prominent details within the constraints of the original kit and provides a solid foundation for expansion should the modeler choose. All by itself it dramatically improves the look of the kit. Set includes three pages of assembly and kit modification instructions.
